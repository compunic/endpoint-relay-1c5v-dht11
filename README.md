Sistem Monitoring Suhu dan Kelembaban
Fitur dan Kelengkapan ESP32
1. Mendeteksi Suhu
2. Mendeteksi Kelembaban
3. Mengirimkan Data Suhu dan kelembaban
4. Mengambil Perintah dari endpoint / Server
5. Menghidupkan dan Mematikan dengan sistem Kontrol Relay
6. Indikator Bunyi Beep Ketika Berhasil Mengirimkan Data Ke server
7. Indikator Lampu LED Menyala Ketika Terhubung dengan Wifi
8. Informasi Ketika Sensor tidak berfungsi

Fitur dan Kelengkapan Endpoint dan Dashboard (SERVER)
1. Menerima data / Informasi Suhu, Kelembaban dan Status Sensor/Relay ON/OFF
2. Menampilkan data yang di Terima dari ESP32 (Perangkat IoT)
3. Menyimpan data yang di terima kedalam Database
4. Memerintahkan atau Mengirimkan informasi ON/OFF ke Perangkat IoT Berdasarkan Data yang di Terima terkini
5. Menampilkan Informasi Dalam periode tertentu dengan fasilitas filter yang ada
6. Menginformasikan Status Relay ON/OFF
